# V2.0 · M0 Migration Dialog Copy

One-time dialog shown on first launch after the `ccl-atlas` → `atomsyn` rename.

## Rationale

- **"搬家" over "迁移"**: 迁移/升级 是工程词汇,搬家 有生活温度,契合"把认知资产拿回来"的叙事。
- **强调备份而非删除**: 正文明确点出 `.ccl-atlas.deprecated.*`,让用户看到退路,降低仪式感带来的紧张。
- **Secondary 不用"取消"**: 用"下次再说"保留用户主权——这是本地优先应用该有的克制。
- **Toast 不用感叹号**: 克制的句号比庆祝的感叹号更符合 Linear/Raycast 的气质。

---

## 中文版(主)

**Title**
搬个新家

**Body**
Atomsyn 是 ccl-atlas 的新名字。我们会把你过去沉淀的原子、实践和技能库搬到新的目录下,原来的文件夹会被重命名为 `.ccl-atlas.deprecated.<时间戳>` 作为备份保留在原处——不会删除,随时可以回去看。

**Primary CTA**
开始搬家

**Secondary CTA**
下次再说

**Success Toast**
你的认知资产已经在新家了。
