```bash

npm run build
export PATH="$HOME/.cargo/bin:$PATH" && (cd src-tauri && cargo check)
```

## 🚨 V2.0 元约束(贯穿 M0-M5)

继承 V1.5 两条元教训 + V2.0 新增三条:

### 继承 V1.5

1. **Tauri v2 deny-by-default + 静默失败** — 任何前端调 Tauri API 没反应,第一反应是查 `src-tauri/capabilities/default.json`
2. **三端数据路径字符级别一致** — M0 改名时这条被放大 10 倍,`vite-plugin-data-api.ts` / `src-tauri/src/lib.rs` / `atomsyn-cli.mjs` 三端的路径字符串必须**完全镜像**,任一处分叉都会让 GUI 和 CLI 各说各话

### V2.0 新增

1. **开放骨架 ≠ 无限蠕变** — 骨架字段值开放,但字段集固定(四维 + tags + insight\_type + context),不在 V2.0 内扩展核心维度
2. **LLM 调用不能成为 UI 路径的同步依赖** — 任何可能让用户等待 LLM 的地方(聚类、ingest、语义对齐)必须有 loading 态 + 可取消 + 失败重试
3. **相遇不炫技** — M4 的 UI 呈现保持克制,📎 标记 + 自然融入文本,**不要弹窗、不要动画强调**、不要喧宾夺主。这是设计哲学不是性能
